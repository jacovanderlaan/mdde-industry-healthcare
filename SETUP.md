# Setup Instructions

## Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `mdde-industry-healthcare`
3. Description: `MDDE models for healthcare - FHIR, EMR, HIPAA`
4. Visibility: **Private**
5. Click "Create repository"

Then push:

```bash
cd c:/Repos/mdde-industry-healthcare
git remote add origin https://github.com/jacovanderlaan/mdde-industry-healthcare.git
git push -u origin main
```
